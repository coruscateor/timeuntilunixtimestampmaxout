# timeuntilunixtimestampmaxout

or

# Time Until Unix Timestamp Max Out

This tool outputs the current Unix timestamp and the remining times until the i64 and u64 maximum values are reached. 

Example output:

```
current unix timestamp: 1669002380

time left until i64::MAX: 9223372035185773427 

time left until u64::MAX: 18446744072040549235
```

Commandline arguments *MAY* be supported in the future.
